By changing the color code in your Sass file you can get the same kind of buttons in any color. All the gradient, border, text colors are calculated automatically, so you only need to provide the base color and you're set.

I strongly recommend not using color names for your CSS classes ("orange" in the example). Use instead semantic names that reflect the meaning of the elements, and attach the colors from CSS under those semantic names.

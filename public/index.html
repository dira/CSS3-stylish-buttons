Sass super-mixins for stylish buttons

Yesterday I found the super yummy pure-CSS buttons designed by Web Designer Wall.

Wondering how one could use this design wisdom despite not posessing the art of subtle gradients and matching colors, I created code that does this for you by magic. And by magic I mean Sass with mixins and the new powerful color functions from version 3.0.

This means that you are not restricted to the (wonderful) colors for which you have the CSS from the Web Designer Wall tutorial...
